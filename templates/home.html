Smile Classifier Website

This website intended to classify an image to smiling or non-smiling. Here you can upload an image and get the result of whether the face is smiling or not. Our Smile Classifier model is built to identify facial expressions and categorize images as either "Smiling" or "Not Smiling." It works with a labeled dataset of face images, where the input data is prepared by resizing the images to 64x64 pixels and converting the labels into numbers for processing. Using Python tools like OpenCV and scikit-learn, we developed and trained the model to perform with high accuracy in detecting smiles. This classifier highlights its usefulness in tasks like emotion recognition and facial analysis.
